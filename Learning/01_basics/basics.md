# C++ Basics: Structure and Output

Every C++ program follows a specific template. The most important parts are:

1. **#include <iostream>**: This "header" allows us to use input and output functions (like printing text).
2. **int main() { ... }**: This is the "entry point." Everything inside these curly braces is what the computer executes.
3. **std::cout**: This stands for "Character Output." We use it to send text to the screen.
4. **return 0;**: This tells the Operating System that the program finished successfully.

### Compiling
To compile this file:
`g++ example.cpp -o basic_app`
